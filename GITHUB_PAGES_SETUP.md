# GitHub Pages Setup

1. Push this repository to GitHub.
2. In GitHub repository settings, enable **Pages**.
3. Set the source to **Deploy from a branch**.
4. Choose the default branch and the `/docs` folder.
5. After Pages is published, your URLs will look like:
   - `https://lioulai85s05-commits.github.io/Cutout-tool/`
   - `https://lioulai85s05-commits.github.io/Cutout-tool/privacy.html`
   - `https://lioulai85s05-commits.github.io/Cutout-tool/support.html`
6. Paste those URLs into App Store Connect.
