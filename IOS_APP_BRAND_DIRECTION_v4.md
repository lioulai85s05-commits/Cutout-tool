# iOS Brand Direction v4

This is the locked direction after selection.

Selected icon base:

- `Split Stage`

Selected brand principle:

- icon and UI must use the same color family and emotional tone

## 1. Brand Direction

Direction name:

- `Split Stage Harmony`

The app is a general object cutout tool, so the brand should communicate:

- object separation
- selected foreground extraction
- precise cut stage
- warm, premium, modern utility feeling

## 2. Visual Language

We are keeping the selected icon logic:

- one selected object or subject breaking out from a darker plate
- a clear foreground/background split
- warm ivory and coral against a dark neutral plate

This works because it visually expresses:

- identify
- isolate
- lift out

## 3. Locked Color System

These colors are now the canonical bridge between icon and UI:

- Shell Background: `#FFF5EB`
- Shell Surface: `#FFF9F1`
- Shell Surface Strong: `#FFF0E2`
- Ink Primary: `#171412`
- Ink Secondary: `#6D625A`
- Border Soft: `#E3D0C0`
- Canvas Dark: `#0B0C0E`
- Canvas Elevated: `#1D1E22`
- Accent Primary: `#FF8F69`
- Accent Deep: `#E6573D`
- Accent Soft: `#FFD8C8`

Rule:

- the UI must not drift into blue tech mode
- the UI must not feel purple or generic AI
- the icon warmth and the app shell warmth must feel related

## 4. UI Translation

How the icon translates into the app:

- warm shell for the overall app
- dark preview and recognition plate inside the editor
- coral accent for active selections and primary actions
- ivory object previews to echo the icon foreground

### Recognition overlay

- dark plate
- multiple colored outlines
- selected instance should get the brand coral emphasis

### Editor

- object sits on a dark canvas
- export controls live on a warm surface
- action buttons use coral gradient or solid coral
